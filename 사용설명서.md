## 1. 프로그램 개요

이 프로그램은 아래 8가지 업무를 자동화합니다.

| 파일명 | 기능 |
|--------|------|
| `email_sender.py` | 고객에게 이메일 자동 발송 |
| `kakao_sender.py` | 카카오 알림톡 자동 발송 |
| `sheets_email_sender.py` | Google Sheets 명단 보고 이메일 일괄 발송 |
| `sheets_dashboard.py` | 매출 현황 대시보드 자동 업데이트 |
| `commerce_automation.py` | 스마트스토어/쿠팡/카페24 주문 자동 수집 |
| `external_data_collector.py` | 날씨/환율/뉴스 자동 수집 |
| `work_tool_integration.py` | Slack 알림/Notion 기록/Google 캘린더 연동 |
| `competitor_crawler.py` | 경쟁사 가격 및 리뷰 자동 수집 |

## 2. 설치 방법

### 2-1. Python 설치 확인
윈도우 검색창에서 `cmd` 검색 → 명령 프롬프트 실행 후 아래 입력:
```
python --version
```
- 버전 숫자(예: `Python 3.11.0`)가 나오면 → 설치 완료, 다음 단계로 이동
- `'python'은 인식되지 않는 명령어` 오류가 나오면 → Python 설치 필요

**Python 설치 방법:**
1. `python.org/downloads` 접속
2. `Download Python` 버튼 클릭
3. 설치 시 반드시 **"Add Python to PATH"** 체크박스 선택
4. Install Now 클릭

---

### 2-2. 필요한 패키지 설치
명령 프롬프트에서 아래 명령어를 한 줄씩 실행합니다:
```
pip install requests
pip install python-dotenv
pip install gspread
pip install google-auth
pip install google-api-python-client
pip install schedule
```

설치 완료 시 `Successfully installed ...` 메시지가 표시됩니다.

---

### 2-3. 프로그램 파일 위치 확인
모든 파일이 같은 폴더 안에 있어야 합니다:
```
내 컴퓨터 > 폴더명
├── .env
├── credentials.json
├── email_sender.py
├── kakao_sender.py
└── (나머지 파일들...)
```

## 3. API 키 발급 방법

### 3-1. Gmail 앱 비밀번호
이메일 자동 발송에 필요합니다.

1. `myaccount.google.com` 접속 (본인 구글 계정으로 로그인)
2. 왼쪽 메뉴 → **보안** 클릭
3. **2단계 인증** 클릭 → 아직 설정 안 됐다면 먼저 설정
4. 2단계 인증 화면 맨 아래 → **앱 비밀번호** 클릭
5. 앱 선택: `메일` / 기기 선택: `Windows 컴퓨터` → **생성** 클릭
6. 화면에 표시된 **16자리 비밀번호** 복사 → `.env` 파일에 입력

> ⚠️ 앱 비밀번호는 한 번만 표시됩니다. 반드시 복사해두세요.

---

### 3-2. 네이버 API (쇼핑 검색 + 뉴스)
경쟁사 가격 수집 및 뉴스 수집에 필요합니다.

1. `developers.naver.com` 접속 → 네이버 계정으로 로그인
2. 상단 메뉴 **Application** → **애플리케이션 등록** 클릭
3. 애플리케이션 이름 입력 (예: `업무자동화`)
4. 사용 API 선택: **검색** 체크 (쇼핑, 뉴스 포함)
5. 환경 추가: `WEB 설정` → URL `http://localhost` 입력
6. 등록 완료 후 **Client ID**와 **Client Secret** 복사 → `.env` 파일에 입력

---

### 3-3. 쿠팡 파트너스 API
쿠팡 주문 수집 및 상품 검색에 필요합니다.

1. `partners.coupang.com` 접속 → 쿠팡 계정으로 로그인
2. 상단 메뉴 **계정** → **API 키 관리** 클릭
3. **API 키 발급** 버튼 클릭
4. **Access Key**와 **Secret Key** 복사 → `.env` 파일에 입력
5. **Vendor ID**는 동일 화면에서 확인 가능

---

### 3-4. 카카오 알림톡
카카오 메시지 자동 발송에 필요합니다.

1. `business.kakao.com` 접속 → 카카오 계정으로 로그인
2. **카카오톡 채널** → **채널 만들기** 클릭
3. 채널 개설 완료 후 → **알림톡** 메뉴 클릭
4. **알림톡 신청** → 사업자 정보 입력 후 신청
5. 심사 승인 후 (보통 2~3일 소요) → **발신 프로필 키** 및 **API 키** 확인
6. **템플릿 등록** → 보낼 메시지 형식 작성 후 심사 요청
7. 템플릿 승인 후 → **템플릿 코드** 확인 → `.env` 파일에 입력

---

### 3-5. Google Sheets / Google Calendar
매출 대시보드 및 캘린더 연동에 필요합니다.

1. `console.cloud.google.com` 접속 → 구글 계정으로 로그인
2. 상단 **프로젝트 선택** → **새 프로젝트** 클릭 → 이름 입력 후 생성
3. 왼쪽 메뉴 → **API 및 서비스** → **라이브러리** 클릭
4. `Google Sheets API` 검색 → **사용 설정**
5. `Google Calendar API` 검색 → **사용 설정**
6. 왼쪽 메뉴 → **사용자 인증 정보** → **사용자 인증 정보 만들기** → **서비스 계정**
7. 서비스 계정 이름 입력 → 완료
8. 생성된 서비스 계정 클릭 → **키** 탭 → **키 추가** → **JSON** 선택
9. 다운로드된 JSON 파일을 프로그램 폴더에 `credentials.json` 이름으로 저장
10. Google Sheets에서 해당 시트를 서비스 계정 이메일에 **공유** (편집자 권한)
11. 시트 URL에서 ID 복사 → `.env` 파일에 입력
    - URL 예시: `docs.google.com/spreadsheets/d/`**여기가ID**`/edit`

---

### 3-6. Slack
Slack 채널 알림 발송에 필요합니다.

1. `api.slack.com/apps` 접속 → Slack 계정으로 로그인
2. **Create New App** → **From scratch** 클릭
3. 앱 이름 입력 → 워크스페이스 선택 → **Create App**
4. 왼쪽 메뉴 **Incoming Webhooks** → 토글 **On**으로 변경
5. **Add New Webhook to Workspace** → 알림 받을 채널 선택
6. 생성된 **Webhook URL** 복사 → `.env` 파일에 입력

---

### 3-7. Notion
업무일지 자동 기록에 필요합니다.

1. `notion.so/my-integrations` 접속 → Notion 계정으로 로그인
2. **새 통합 만들기** 클릭
3. 이름 입력 → 워크스페이스 선택 → **제출**
4. 생성된 **내부 통합 토큰** 복사 → `.env` 파일에 입력
5. Notion에서 사용할 데이터베이스 페이지 접속
6. 오른쪽 상단 `···` → **연결 추가** → 방금 만든 통합 선택
7. 데이터베이스 URL에서 ID 복사 → `.env` 파일에 입력
    - URL 예시: `notion.so/`**여기가ID**`?v=...`

---

### 3-8. OpenWeather (날씨)
날씨 자동 수집에 필요합니다.

1. `openweathermap.org` 접속 → **Sign Up** 클릭
2. 이메일/비밀번호 입력 후 가입 (무료)
3. 가입 확인 이메일 클릭
4. 로그인 후 → 상단 이름 클릭 → **My API Keys**
5. 기본 생성된 **API Key** 복사 → `.env` 파일에 입력

---

### 3-9. ExchangeRate (환율)
환율 자동 수집에 필요합니다.

1. `exchangerate-api.com` 접속 → **Get Free Key** 클릭
2. 이메일 입력 후 가입 (무료, 월 1,500회 제공)
3. 이메일 인증 완료
4. 대시보드에서 **API Key** 복사 → `.env` 파일에 입력

---

### 3-10. 인스타그램 (Meta Graph API)
인스타그램 게시물 자동 발행 및 통계 조회에 필요합니다.

> ⚠️ **비즈니스 계정 또는 크리에이터 계정**이어야 합니다. 개인 계정은 API 사용 불가.

1. `developers.facebook.com` 접속 → 페이스북 계정으로 로그인
2. **My Apps** → **Create App** 클릭
3. 앱 유형: **Business** 선택 → 앱 이름 입력 → 생성
4. 왼쪽 메뉴 → **Instagram Graph API** → **설정** 클릭
5. 인스타그램 비즈니스 계정 연결
6. **사용자 토큰 생성** → 권한 선택: `instagram_basic`, `instagram_content_publish`
7. 생성된 **Access Token** 복사 → `.env`의 `IG_ACCESS_TOKEN`에 입력
8. 토큰 생성 화면에서 **User ID** 확인 → `.env`의 `IG_USER_ID`에 입력

> ⚠️ 기본 토큰은 60일 후 만료됩니다. 장기 토큰으로 교환하거나 주기적으로 갱신하세요.

---

### 3-11. 트위터/X (Twitter API v2)
트윗 자동 발행 및 성과 조회에 필요합니다.

> ⚠️ 트위터 API는 **유료 플랜(Basic, $100/월)** 이상에서 쓰기 기능이 가능합니다. 무료 플랜은 읽기만 가능합니다.

1. `developer.twitter.com` 접속 → 트위터 계정으로 로그인
2. **Sign up for Free Account** 클릭 → 용도 설명 입력 후 가입
3. **Dashboard** → **+ Create Project** 클릭
4. 프로젝트 이름 입력 → 앱 환경 선택 → **Create App**
5. **Keys and Tokens** 탭 클릭
6. **API Key and Secret** → **Generate** → 복사 → `.env`에 입력
7. **Access Token and Secret** → **Generate** → 복사 → `.env`에 입력

> ⚠️ API Key, Secret, Access Token, Access Token Secret 4가지 모두 필요합니다.
> ⚠️ 한 번만 표시되므로 반드시 즉시 복사해두세요.

---

### 3-12. 유튜브 (YouTube Data API v3)
유튜브 채널 통계 및 경쟁 채널 분석에 필요합니다.

1. `console.cloud.google.com` 접속 → 구글 계정으로 로그인
2. 기존 프로젝트 선택 또는 새 프로젝트 생성
3. 왼쪽 메뉴 → **API 및 서비스** → **라이브러리** 클릭
4. `YouTube Data API v3` 검색 → **사용 설정** 클릭
5. 왼쪽 메뉴 → **사용자 인증 정보** → **사용자 인증 정보 만들기** → **API 키**
6. 생성된 **API 키** 복사 → `.env`의 `YOUTUBE_API_KEY`에 입력
7. 내 채널 ID 확인:
   - 유튜브 접속 → 프로필 클릭 → **YouTube 스튜디오**
   - 왼쪽 메뉴 → **설정** → **채널** → **고급 설정**
   - **채널 ID** 복사 → `.env`의 `YOUTUBE_CHANNEL_ID`에 입력

> ℹ️ YouTube Data API는 하루 **10,000 유닛 무료** 제공됩니다. 일반적인 분석 용도로는 충분합니다.

## 4. API 키 입력 방법

프로그램 폴더에서 `.env` 파일을 메모장으로 열고 각 항목에 발급받은 키를 입력합니다.

```
GMAIL_USER=내이메일@gmail.com
GMAIL_PASSWORD=abcd efgh ijkl mnop
NAVER_CLIENT_ID=발급받은ID
NAVER_CLIENT_SECRET=발급받은SECRET
...
```

> ⚠️ `=` 앞뒤에 띄어쓰기 없이 입력하세요.
> ⚠️ 따옴표(`"`) 없이 입력하세요.

### 멀티 SNS 동시 발행
```
python sns_multi_post.py
```
- 파일 내 `text`, `image_url`, `platforms` 수정 후 실행
- `platforms` 예시: `["instagram", "twitter", "slack"]`
- 인스타그램은 `image_url` 필수, 없으면 자동으로 건너뜁니다

---

### 콘텐츠 예약 스케줄러
```
python content_scheduler.py
```
- Google Sheets `콘텐츠스케줄` 시트 구성: `날짜 | 시간 | 내용 | 이미지URL | 플랫폼 | 발행여부`
- 날짜 형식: `2026-06-27` / 시간 형식: `09:00`
- 플랫폼 여러 개: `twitter,instagram` (쉼표로 구분)
- 실행 중 컴퓨터를 끄면 중단됩니다

---

### 유튜브 채널 분석
```
python youtube_analytics.py
```
- 실행 시 내 채널 통계 + 최근 영상 성과 + 경쟁 채널 분석 출력
- 결과는 `youtube_report_날짜.csv`로 저장
- 경쟁 채널 키워드 변경: 파일 내 `search_competitor_channels("요리 브이로그")` 수정
- 주간 리포트 이메일 발송: 파일 하단 주석 제거

## 6. 자동 실행 설정

매일 정해진 시간에 자동으로 실행하려면 아래 파일을 만들어 실행하세요.

**`scheduler.py` 파일 생성 후 실행:**
```python
import schedule
import time
from commerce_automation import run_all as commerce
from work_tool_integration import morning_briefing
from external_data_collector import run_all as data_collect

schedule.every().day.at("08:00").do(morning_briefing)   # 모닝 브리핑
schedule.every().day.at("09:00").do(commerce)           # 주문 수집
schedule.every().day.at("09:10").do(data_collect)       # 날씨/환율/뉴스

while True:
    schedule.run_pending()
    time.sleep(60)
```

```
python scheduler.py
```

> 컴퓨터가 켜져 있는 동안 계속 실행됩니다.
> 컴퓨터를 끄면 중단됩니다.

## 7. 자주 묻는 질문

**Q. 실행하면 오류가 납니다.**
- `.env` 파일에 API 키가 올바르게 입력됐는지 확인하세요
- `=` 앞뒤 띄어쓰기, 따옴표가 없는지 확인하세요

**Q. 이메일이 발송되지 않습니다.**
- Gmail 앱 비밀번호가 맞는지 확인하세요
- 일반 비밀번호가 아닌 **앱 비밀번호**를 입력해야 합니다

**Q. Google Sheets에 접근이 안 됩니다.**
- `credentials.json` 파일이 프로그램 폴더 안에 있는지 확인하세요
- 해당 시트가 서비스 계정 이메일에 **공유**됐는지 확인하세요

**Q. 카카오 알림톡 발송이 안 됩니다.**
- 템플릿이 **승인 완료** 상태인지 확인하세요 (미승인 템플릿은 발송 불가)

**Q. 인스타그램 발행이 안 됩니다.**
- 비즈니스 또는 크리에이터 계정인지 확인하세요
- Access Token이 만료(60일)됐을 수 있습니다 — 재발급 후 `.env` 업데이트
- 이미지 URL이 공개 접근 가능한지 확인하세요 (로그인 없이 열려야 함)

**Q. 트위터 발행 시 403 오류가 납니다.**
- 트위터 API **Basic 플랜($100/월)** 이상이어야 쓰기가 가능합니다
- 무료 플랜은 읽기 전용입니다

**Q. 유튜브 API 한도 초과 오류가 납니다.**
- 하루 10,000 유닛을 모두 사용한 상태입니다
- 다음 날 자정(태평양 시간, 한국 기준 오후 4시)에 자동 초기화됩니다
- 너무 자주 실행하지 말고 하루 1~2회로 제한하세요

**Q. 콘텐츠 스케줄러가 정해진 시간에 발행이 안 됩니다.**
- 프로그램이 실행 중인지 확인하세요 (터미널 창이 열려 있어야 합니다)
- Google Sheets 시트명이 정확히 `콘텐츠스케줄`인지 확인하세요
- 날짜/시간 형식이 `2026-06-27`, `09:00` 형식인지 확인하세요

**Q. 멀티 SNS 발행 시 일부 플랫폼만 성공합니다.**
- 각 플랫폼 API 키가 모두 `.env`에 입력됐는지 확인하세요
- 실패한 플랫폼의 오류 메시지를 확인 후 개발자에게 문의하세요

**Q. 스케줄러(`scheduler.py`)를 실행했는데 아무것도 안 됩니다.**
- 정상입니다 — 등록된 시간이 될 때까지 대기 중입니다
- 터미널 창을 닫지 말고 그대로 두세요

## 8. 주의사항

### 보안 공통
- `.env` 파일과 `credentials.json` 파일은 **절대 타인에게 전달하지 마세요**
- API 키가 노출되면 본인 계정으로 무단 사용 및 과금이 발생할 수 있습니다
- 이 파일들을 카카오톡, 이메일, USB 등으로 공유하지 마세요

---

### 파일별 주의사항

#### 이메일 (`email_sender.py`, `sheets_email_sender.py`)
- Gmail은 하루 **최대 500건** 발송 제한 (무료 계정 기준)
- 대량 발송 시 스팸으로 분류될 수 있으니 한 번에 너무 많이 보내지 마세요
- 앱 비밀번호는 2단계 인증이 켜져 있어야 발급 가능합니다
- 앱 비밀번호를 재발급하면 기존 비밀번호는 자동 만료됩니다

#### 카카오 알림톡 (`kakao_sender.py`)
- **승인된 템플릿** 내용과 정확히 일치해야 발송됩니다 (한 글자라도 다르면 실패)
- 템플릿 심사는 보통 **2~3 영업일** 소요됩니다
- 광고성 메시지는 반드시 **수신 동의**를 받은 고객에게만 발송해야 합니다 (정보통신망법)
- 야간(오후 9시 ~ 오전 8시)에는 광고성 메시지 발송 금지

#### Google Sheets / 대시보드 (`sheets_email_sender.py`, `sheets_dashboard.py`)
- Google Sheets API는 분당 **60회** 요청 제한이 있습니다
- 한 번에 너무 많은 행을 처리하면 오류가 날 수 있으니 1,000행 이상은 나눠서 실행하세요
- 서비스 계정 이메일에 시트를 **공유(편집자 권한)** 해야 쓰기가 가능합니다

#### 커머스 자동화 (`commerce_automation.py`)
- 스마트스토어 API는 **판매자 계정**에서만 발급 가능합니다
- 쿠팡 API는 **쿠팡 파트너스 계정**이 필요합니다 (일반 구매자 계정 불가)
- 카페24는 쇼핑몰 운영 중인 계정에서만 발급 가능합니다
- 각 플랫폼 API는 **하루 호출 횟수 제한**이 있으니 너무 자주 실행하지 마세요

#### 날씨/환율/뉴스 (`external_data_collector.py`)
- OpenWeather 무료 플랜: 분당 **60회**, 월 **1,000,000회** 제한
- ExchangeRate 무료 플랜: 월 **1,500회** 제한 — 매분 실행하면 이틀 만에 소진됩니다
- 네이버 검색 API: 하루 **25,000회** 제한

#### Slack/Notion/캘린더 (`work_tool_integration.py`)
- Slack 무료 플랜은 메시지 **90일치**만 보관됩니다
- Notion API는 분당 **3회** 요청 제한이 있습니다
- Google Calendar 서비스 계정은 본인 캘린더에 **공유 설정**이 필요합니다
  - 캘린더 앱 → 내 캘린더 → 설정 → 특정 사용자와 공유 → 서비스 계정 이메일 추가

#### 경쟁사 가격 수집 (`competitor_crawler.py`)
- 네이버 쇼핑 API: 하루 **25,000회** 제한
- 쿠팡 파트너스 API: 상품 정보는 **파트너스 링크 형태**로만 제공됩니다
- 수집한 데이터를 **외부에 재배포하거나 상업적으로 이용**하면 약관 위반입니다
- 개인 분석 및 내부 참고용으로만 사용하세요

#### 인스타그램 (`instagram_scheduler.py`)
- **비즈니스 또는 크리에이터 계정**에서만 API 사용 가능합니다
- Access Token은 **60일 후 만료**됩니다 — 만료 전에 갱신하세요
- 하루 게시물 발행 횟수 제한: **50개**
- 이미지는 반드시 **공개 접근 가능한 URL** 형식이어야 합니다

#### 트위터/X (`twitter_automation.py`)
- 트윗 **쓰기 기능**은 Basic 플랜($100/월) 이상에서만 가능합니다
- 무료 플랜은 읽기 전용입니다
- 트윗은 **280자** 제한입니다 (한글은 1자당 2자로 계산됩니다)
- 같은 내용을 반복 트윗하면 **스팸으로 계정이 정지**될 수 있습니다

#### 멀티 SNS 동시 발행 (`sns_multi_post.py`)
- 각 플랫폼의 이미지 규격이 다릅니다:
  - 인스타그램: 정사각형(1:1) 또는 세로형(4:5) 권장
  - 트위터: 16:9 비율 권장
- 인스타그램은 **이미지 없이 텍스트만 발행 불가**합니다
- 발행 실패 시 오류 내용을 확인하고 개발자에게 문의하세요

#### 콘텐츠 스케줄러 (`content_scheduler.py`)
- 스케줄러 실행 중 **컴퓨터를 끄면 발행이 중단**됩니다
- 장시간 자동 실행이 필요하다면 개발자에게 **서버 운영** 형태로 전환을 문의하세요
- Google Sheets 시트명이 정확히 `콘텐츠스케줄`이어야 합니다

#### 유튜브 분석 (`youtube_analytics.py`)
- YouTube Data API 무료 한도: 하루 **10,000 유닛**
  - 채널 정보 조회: 1회당 1 유닛
  - 영상 목록 조회: 1회당 100 유닛
  - 한도 초과 시 당일 자정(태평양 시간)에 초기화됩니다
- 타인의 채널을 분석하는 것은 **공개된 정보 범위 내**에서만 가능합니다
- 주간 리포트는 **매주 1회** 실행을 권장합니다 (유닛 절약)
