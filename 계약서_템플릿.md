## 4. 의뢰인 의무

- API 키는 의뢰인 본인 명의로 직접 발급합니다.
- 납품받은 .env 파일(API 키 파일)을 제3자에게 전달하지 않습니다.
- 마케팅 수신 동의를 받은 고객에게만 자동 발송을 사용합니다.
- 납품 완료 후 API 키를 재발급합니다.

---

## 5. 수탁인 의무 및 면책

- 납품 파일이 안내대로 실행했음에도 작동하지 않으면 무상으로 수정합니다.
- 아래 경우는 수탁인의 책임 범위에서 제외됩니다.
  - API 키 미발급 또는 잘못 입력으로 인한 오류
  - 의뢰인 PC 환경(Python 버전, 백신, 방화벽)으로 인한 오류
  - 플랫폼 정책 변경으로 API가 중단된 경우
  - 최초 작업 범위에 없던 기능 추가 요청

---

## 6. 개인정보 처리

- 수탁인은 개인정보 처리 수탁자로서 의뢰인으로부터 제공받은 고객 데이터를 납품 목적 외에 사용하지 않습니다.
- 수탁인은 납품 완료 후 의뢰인의 고객 데이터를 즉시 삭제합니다.
- 개인정보 유출 사고 발생 시 즉시 의뢰인에게 통보합니다.
