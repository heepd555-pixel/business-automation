# DART 재무 스크리너 사용설명서

---

## 이 프로그램이 하는 일

금융감독원 DART 공식 API를 통해 전체 상장사(KOSPI+KOSDAQ)의
재무제표를 자동 수집하고, 원하는 조건으로 필터링해 결과를 저장합니다.

> ⚠ 투자 추천 도구가 아닙니다. 재무 데이터 필터링 도구이며
> 투자 판단은 본인 책임 하에 이루어져야 합니다.

---

## 1. 사전 준비

### 1-1. DART API 키 발급 (무료)

1. https://opendart.fss.or.kr 접속
2. 회원가입 → 로그인
3. 마이페이지 → API 신청 → 즉시 발급
4. 발급받은 API 키 복사

### 1-2. .env 파일 설정

`.env.example` 파일을 복사해서 `.env`로 저장 후 아래 항목 입력

```
DART_API_KEY=발급받은키입력

# 이메일 리포트 사용 시
GMAIL_USER=본인이메일@gmail.com
GMAIL_PASSWORD=앱비밀번호

# Google Sheets 사용 시
GOOGLE_SHEET_ID=시트ID
GOOGLE_CREDENTIALS_FILE=credentials.json
```

### 1-3. 패키지 설치

```
pip install requests python-dotenv gspread google-auth
```

---

## 2. 필터 조건 설정

`dart_설정.json` 파일을 메모장으로 열어서 수정합니다.

```json
"필터조건": {
    "ROE_최소(%)": 10,          ← ROE 10% 이상인 기업만
    "부채비율_최대(%)": 150,    ← 부채비율 150% 이하인 기업만
    "매출성장률_최소(%)": 5,    ← 전년 대비 매출 5% 이상 성장
    "영업이익률_최소(%)": 5,    ← 영업이익률 5% 이상
    "매출액_최소(억원)": 100    ← 매출액 100억 이상 기업만
}
```

조건을 낮추면 더 많은 기업이, 높이면 더 적은 기업이 나옵니다.

---

## 3. 출력 설정

```json
"출력설정": {
    "CSV저장": true,                    ← true: CSV 파일 저장
    "Google_Sheets_업데이트": false,    ← true: 구글 시트 업데이트
    "이메일_리포트_발송": false,         ← true: 이메일 발송
    "최대_출력_기업수": 50              ← 최대 50개까지 출력
}
```

## 5. 결과 확인

실행 완료 후 같은 폴더에 CSV 파일이 생성됩니다.

```
스크리닝_2024_0701_1430.csv
```

| 컬럼 | 설명 |
|---|---|
| 기업명 | 상장사 이름 |
| 종목코드 | 증권사 앱에서 검색 시 사용 |
| ROE(%) | 자기자본이익률 |
| 부채비율(%) | 낮을수록 재무 안정적 |
| 영업이익률(%) | 높을수록 수익성 좋음 |
| 매출성장률(%) | 전년 대비 매출 증가율 |
| 매출액(억) | 연간 매출액 |
| 영업이익(억) | 연간 영업이익 |
| 순이익(억) | 연간 당기순이익 |

## 6. 자주 묻는 질문

**Q. 실행하면 오류가 나요.**
A. .env 파일에 DART_API_KEY가 입력됐는지 확인하세요.

**Q. 결과가 0개예요.**
A. 필터 조건이 너무 엄격합니다. dart_설정.json에서 조건을 낮춰보세요.

**Q. 시간이 너무 오래 걸려요.**
A. 상장사가 2,000개 이상이라 30분~1시간 소요될 수 있습니다.

**Q. 매년 실행해야 하나요?**
A. 네. 분석_기준연도를 매년 업데이트하고 실행하면 됩니다.

**Q. 이 결과를 믿고 투자해도 되나요?**
A. 재무 데이터 필터링 결과일 뿐입니다. 투자 판단은 반드시
   추가 분석 후 본인 책임 하에 결정하세요.

## 7. 법적 고지

- DART 공식 API를 사용하며 데이터 수집은 합법입니다.
- 수집 데이터는 내부 분석 전용으로만 사용하세요.
- 본 프로그램은 투자자문업에 해당하지 않습니다.
- 투자로 인한 손실에 대해 개발자는 책임지지 않습니다.
